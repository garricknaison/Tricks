# Tricks

This is an interactive text puzzle game that uses clever logic riddles and classic games to try to outsmart you. Pay close attention to everything on your screen! All the best!

## Program Summary
A major mechanic of the game is its strict progress system: while the early stages allow multiple attempts, the later stages will completely shut down the application if a single wrong answer is entered, forcing the player to restart the entire game from the beginning.

## Key Features
* **Multi-Stage Progression:** Players must navigate through various types of challenges, including math problems, logic riddles, and memory tests.
* **Tic-Tac-Toe AI Engine:** Features a fully functional Tic-Tac-Toe game where a 3x3 grid is managed by a 2D array. The player takes turns placing marks against an automated system opponent that calculates its moves using random grid selection.
* **Scavenger Hunt Element:** The program utilizes a hidden alphanumeric code. The clues to piece this code together are subtly revealed to the player as they successfully pass through earlier stages of the game.
* **Paced Output:** The game utilizes timed delays between printed text lines to control the reading pace and build tension during gameplay.
